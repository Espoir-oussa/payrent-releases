    Télécharger l'APK Voir la release

    Mode opératoire (manuel)

    1. Construis l'APK localement : flutter build apk --release.
    2. Crée une Release GitHub dans le repository utilisé pour héberger les APKs et upload l'APK en tant qu'asset.
    3. Dans le dépôt de la landing page (`tmp-payrent-releases`), mets à jour version.json (ou utilise le script scripts/publish_apk.ps1 fourni) puis pousse sur la branche main.
    4. La page se mettra à jour automatiquement via GitHub Pages.
